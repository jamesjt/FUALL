# That which is below is like that which is above and that which is above is

"That which is below is like that which is above and that which is above is like that which is below, to do the miracles of one only thing."This quote, found in Isaac Newton’s translation as noted in Wikipedia - Emerald Tablet, encapsulates the principle of "as above, so below," highlighting the correspondence between the microcosm and macrocosm. It suggests that truth is revealed through understanding these interconnected patterns, a process that requires study, contemplation, and transformation over time. This aligns with the alchemical process, where the philosopher’s stone (or ultimate truth) is achieved through stages that unfold temporally.

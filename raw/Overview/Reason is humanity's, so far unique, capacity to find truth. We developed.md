# Reason is humanity's, so far unique, capacity to find truth. We developed

Reason is humanity's, so far unique, capacity to find truth. We developed methods to justify our beliefs about Reality. As such we started acquiring some knowledge/Scientia/science. 
There is much work left to be done in expanding these categories of knowledge or 'the Sciences' but we've made great strides toward understanding Reality, and in doing so improved our ability to Reason, as we can take notice if when our current tools for finding truth don't quite get us to reality so we then work to improve them. Our best methods of acquiring knowledge we refer to as our scientific methods, and we've added various tools to these methods over time and will keep finding more to add.
This allowed us to start down the path of answering one of our fundamental questions. What is existence and how does it work?[2]
